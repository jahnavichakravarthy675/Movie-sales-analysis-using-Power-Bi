Project Overview

The movie industry generates large volumes of data related to genres, ratings, revenue, audience engagement, and critical reviews. Extracting meaningful insights from this data is essential for understanding performance trends and audience preferences.

This project analyzes IMDb-based movie sales data using Power BI to uncover revenue patterns, identify top-performing genres and directors, and evaluate the impact of ratings, votes, and runtime on box office success.

Objectives

Analyze movie performance across years, genres, and directors.

Identify revenue trends and audience preferences.

Study the impact of ratings, votes, and runtime on movie success.

Support data-driven decision-making in the entertainment industry.
